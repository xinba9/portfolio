# 如何创建Hermes Agent桌面快捷方式

由于系统安全限制，我无法直接创建Windows .lnk快捷方式文件。

但我已经为你创建了 **批处理文件启动器**，它在功能上等同于快捷方式：

## ✅ 已创建的启动文件

### 文件1：Hermes Agent.bat
- **位置**：`C:\Users\EDY\Desktop\Hermes Agent.bat`
- **使用方法**：**双击即可启动Hermes聊天**
- **功能**：自动配置环境，启动Hermes

### 文件2：Hermes Chat.bat（之前的版本）
- **位置**：`C:\Users\EDY\Desktop\Hermes Chat.bat`

---

## 🎯 如何使用

### 方法1：直接双击批处理文件（推荐）⭐

1. 打开桌面
2. 找到 **"Hermes Agent.bat"** 文件
3. **双击它**
4. 会打开一个命令行窗口并启动Hermes聊天

### 方法2：手动创建真正的快捷方式（可选）

如果你想要一个"真正的"Windows快捷方式（.lnk文件），可以手动创建：

#### 步骤：
1. 右键点击桌面 → **新建** → **快捷方式**
2. 在"对象位置"输入：
   ```
   C:\Windows\System32\cmd.exe /k "C:\Users\EDY\Desktop\Hermes Agent.bat"
   ```
3. 点击**下一步**
4. 名称输入：**Hermes Agent**
5. 点击**完成**
6. （可选）右键新快捷方式 → **属性** → 更改图标

---

## 📋 批处理文件功能说明

`Hermes Agent.bat` 文件内容：

```batch
@echo off
chcp 65001 >nul  # 设置UTF-8编码
echo 正在启动Hermes Agent...
echo.

REM 添加Hermes到PATH
set PATH=C:\Users\EDY\.workbuddy\binaries\python\versions\3.13.12\Scripts;%PATH%

REM 启动Hermes聊天
hermes chat

pause
```

## 💡 两种启动方式对比

| 方式 | 优点 | 缺点 |
|------|------|------|
| **批处理文件(.bat)** | ✅ 已创建，可直接使用<br>✅ 可以看到启动信息<br>✅ 退出后有pause提示 | 图标是批处理文件的默认图标 |
| **快捷方式(.lnk)** | ✅ 可以自定义图标<br>✅ 看起来更"正式" | 需要手动创建 |

## 📝 注意事项

1. **杀毒软件提示**：第一次运行时，杀毒软件可能会提示，选择"允许"即可
2. **命令行窗口**：会打开一个黑色命令行窗口，这是正常的
3. **退出方法**：在Hermes中输入 `/exit` 或按 `Ctrl+C`
